## 2. 新規アセットの追加・統合手順

今後、新しい天体の実写拡大写真を追加する際は、以下のステップに従って実装を行ってください。

### Step 1: 画像アセットの準備と配置
1. 追加したい天体の高解像度画像を用意します。
   * **推奨形式**: PNG形式（透過、または背景が純黒 `#000000`）
   * **アスペクト比**: 1:1 (正方形)
2. 用意した画像ファイルを `front/public/assets/` ディレクトリに小文字のファイル名（例: `ngc1234.png`）で配置します。

### Step 2: `DSO_FIXED_COORDS` への基本座標の定義
フロントエンドで直接天体座標を参照できるようにするため、`front/src/main.ts` 上部の `DSO_FIXED_COORDS` オブジェクトに天体の赤道座標を追加します。
* 値は赤経 (`ra`、時間換算 $0 \sim 24$)、赤緯 (`dec`、度数 $-90 \sim 90$) で指定します。

```typescript
const DSO_FIXED_COORDS: Record<string, { ra: number; dec: number }> = {
  // ...既存の定義...
  NGC1234: { ra: 12.345, dec: -6.789 }, // 新規追加
};
```

### Step 3: `buildDsoPhotos()` へのロード設定の追加
`front/src/main.ts` 内の `buildDsoPhotos()` 関数にある `dsoConfigs` 配列に、アセットのロード設定を記述します。

```typescript
function buildDsoPhotos() {
  const loader = new THREE.TextureLoader();
  const dsoConfigs = [
    // ...既存の設定...
    { id: 'NGC1234', file: 'ngc1234.png', ra: 12.345, dec: -6.789, scale: 15.0 } // 新規追加
  ];
  
  dsoConfigs.forEach(cfg => {
    // 自動的に SpriteMaterial としてロードされ、dsoPhotoObjects に格納されます。
  });
}
```
* `scale`: 望遠鏡の接眼レンズ（アイピース）内で、どれくらいの大きさで表示させるかのスケール値（3D空間サイズ）です。見かけの大きさ（角度）に合わせて適宜調整してください。

### Step 4: `updateDsoPhotos()` への描画・追尾バインドの追加
天体がターゲット指定されて追尾中（GoTo完了後）に、3D空間上の正しい位置に追随して写真を描画・フェードインさせるため、`updateDsoPhotos()` 内の `dsoConfigs` リストへ登録します。

```typescript
function updateDsoPhotos(lst: number) {
  // ...中略...
  // 3. DSO天体の更新
  const dsoConfigs = [
    { id: 'M31', ...DSO_FIXED_COORDS['M31'] },
    // ...
    { id: 'NGC1234', ...DSO_FIXED_COORDS['NGC1234'] }, // 新規追加
  ];

  dsoConfigs.forEach(cfg => {
    // ターゲット指定時にカメラのズーム(FOV)に応じて不透明度がフェードインします。
  });
}
```

### Step 5: 自動導入 (GoTo) UI への反映
ターゲットセレクトボックスから選択できるようにするため、`front/index.html` 内の `<select id="obs-target-select">` に `<option>` を追加します。

```html
<select id="obs-target-select">
  <option value="none">-- ターゲット選択 --</option>
  <!-- ...既存の天体... -->
  <option value="NGC1234">NGC 1234 銀河</option>
</select>
```

## 3. 動作確認とデバッグ
1. フロントエンドとバックエンドを起動し、ブラウザでアプリケーションを開きます。
2. ターゲットのセレクトボックスから追加した天体を選択し、「自動導入」を実行します。
3. 自動的に視野角が望遠鏡モードに切り替わり、カメラが自動的に天体へ向いた後、ズームインすることで実写アセットがフェードインして中央に表示されることを確認します。
4. 背景の星空が透けて邪魔になっていないこと（自動オクルージョン）を確認します。
