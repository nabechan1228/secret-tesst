# Stellaris - Professional Interactive Planetarium

精密な天体計算アルゴリズムと3Dレンダリングを用いた、ウェブブラウザで動くリアルタイム・インタラクティブ・プラネタリウムです。

---

## 🚀 特徴

- **5000個以上のリアルな恒星描画**: HIP番号ベースの星カタログデータから、主要な恒星（5,044個）を3D空間上に配置。
- **B-V色指数によるカラー再現**: 恒星のスペクトルタイプ（色温度）に応じたリアルな色調（青白い星〜赤い星）を再現。
- **精密な天体座標計算 (Python)**: 
  - 観測地点（緯度・経度）と日時（世界時・ローカル時間）に基づく、ユリウス日（JD）および地方恒星時（LST）の算出。
  - 各恒星の赤経（RA）・赤緯（Dec）から、観測者から見た地平座標（高度 Altitude / 方位角 Azimuth）への高精度な動的変換。
- **88星座線・星座情報の統合**: 
  - 全天の88星座を繋ぐ星座線をリアルタイム描画。
  - 星座を選択すると、その星座の学術名、和名、解説、および主要な恒星の情報を閲覧可能。

---

## 🛠️ 技術スタック

### バックエンド (Python)
- **FastAPI**: 軽量・高速なWeb APIフレームワーク。
- **Uvicorn**: ASGIサーバー。
- **天体計算ロジック**: `math`モジュールを利用したフルスクラッチ天体計算（外部天体ライブラリに依存せず、数式に基づくJD/LST/地平変換を実装）。
- **データキャッシュ**: 高速なロードを実現するLRUキャッシュ構造。

### フロントエンド (Vite / TypeScript / Three.js)
- **Three.js**: WebGLを利用した3D描画ライブラリ。恒星は専用のスプライトテクスチャを割り当てた `Points` オブジェクトで超高速描画。
- **Vite**: 高速なモジュールバンドラー。
- **TypeScript**: 静的型付けによる堅牢なフロントエンド実装。
- **CSS**: グラスモルフィズム（背景ぼかし）とネオンエフェクトを採用したサイバーパンク/近未来的UI。

## 🏎️ 起動方法

アプリケーションを起動するには、バックエンドとフロントエンドの両方のサーバーを起動する必要があります。

### 1. バックエンドの起動 (Python)

1. `back` ディレクトリに移動します。
2. 依存ライブラリをインストールします。
   ```bash
   pip install fastapi uvicorn
   ```
3. FastAPIサーバーを起動します。
   ```bash
   uvicorn main:app --host 127.0.0.1 --port 8000 --reload
   ```

*バックエンドAPIは `http://localhost:8000` で稼働します。*

### 2. フロントエンドの起動 (Node.js)

1. `front` ディレクトリに移動します。
2. 依存パッケージをインストールします。
   ```bash
   npm install
   ```
3. 開発用サーバーを起動します。
   ```bash
   npm run dev
   ```

*ブラウザで `http://localhost:3000` にアクセスしてアプリケーションを利用できます。*
